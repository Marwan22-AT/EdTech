🚀 EdTech - Modern Learning Platform

EdTech هي منصة تعليمية متطورة مصممة لتقديم تجربة تعلم سلسة واحترافية، مع التركيز على مهارات المستقبل. تم بناء المنصة مع مراعاة خفض تكاليف التشغيل بنسبة 40% دون المساس بالجودة، من خلال استخدام أحدث التقنيات السحابية والأدوات مفتوحة المصدر.

✨ المميزات الرئيسية (Key Features)

🌍 ثنائية اللغة (Bilingual Support): دعم كامل للغتين العربية والإنجليزية مع تبديل لحظي للواجهات (RTL/LTR).

🔐 نظام دخول ذكي (Smart Auth):

تسجيل الدخول عبر Google و Facebook.

إمكانية الاستكشاف كضيف (Guest Mode) لتجربة الموقع قبل التسجيل.

💳 التوطين المالي (Localization): الأسعار معروضة بالجنيه المصري (EGP) لتناسب السوق المحلي، مع وجود كورسات مجانية بالكامل.

🎨 تجربة مستخدم سلسة (Smooth UX): تنقل سريع وبسيط (Minimalist Design) يقلل من تشتت الطالب.

📱 متوافق مع الموبايل (Responsive): يعمل بكفاءة عالية على جميع أحجام الشاشات.

🛠 التكنولوجيا المستخدمة (Tech Stack)

تم اختيار هذه الأدوات لضمان أعلى أداء بأقل تكلفة (Budget-Friendly):

Frontend: React.js + Tailwind CSS (RTL support).

Localization: React-i18next.

Backend/Auth: Firebase / Supabase (Serverless architecture).

Video Hosting: Optimized Video Embedding (Vimeo/YouTube Unlisted).

Icons: Lucide React / FontAwesome.

💡 استراتيجية خفض الميزانية (Optimization Strategy)

لتحقيق خفض الميزانية بنسبة 40%:

الاعتماد على Serverless Architecture لتقليل تكاليف السيرفرات الشهرية.

استخدام Headless CMS لإدارة المحتوى بسهولة دون الحاجة لبرمجة لوحة تحكم معقدة.

أتمتة إصدار الشهادات بصيغة PDF لتوفير الجهد اليدوي.
