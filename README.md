# 🎓 Ed Tech - منصة الكورسات التعليمية

موقع احترافي لعرض وبيع الكورسات التعليمية مع دعم اللغتين العربية والإنجليزية.

## 🚀 المميزات

- ✅ تسجيل دخول أولي (Landing Page)
- ✅ دعم اللغة العربية (RTL) والإنجليزية (LTR)
- ✅ كورسات مجانية ومدفوعة بأسعار بالجنيه المصري
- ✅ تصنيفات: Digital Marketing, Digital Creator, Game Development, Animation, Game Design
- ✅ تصميم متجاوب (Responsive) لجميع الأجهزة
- ✅ نظام فلترة وبحث للكورسات
- ✅ صفحة تفاصيل الكورس مع محتوى تفاعلي
- ✅ خصم 40% مُظهر على الكورسات المدفوعة
